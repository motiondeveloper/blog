## Objects and their properties

Collection of related data. Hierarchy, e.g. compositions and layers. Variables attached to an object. Key and value.

If value is a function its a method.

## Dot notation

```js
anObject.someProperty;
```

## Bracket notation

Dynamic property keys.

```js
anObject['someProperty'];
```

## Setting object properties

```js
anObject.someProperty = aValue;
```
